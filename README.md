# Gradient Text Effect - CodingLab

Um projeto experimental moderno e minimalista focado na aplicação de efeitos tipográficos avançados utilizando exclusivamente HTML5 e CSS3. O objetivo principal é renderizar um preenchimento em gradiente linear diretamente no corpo do texto, criando um visual sofisticado e dinâmico sem a necessidade de converter as fontes em imagens.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do contêiner e marcação do texto principal.
* CSS3: Estilização tipográfica avançada e manipulação de propriedades de mascaramento.
* Flexbox: Centralização absoluta e responsiva do elemento de texto na viewport.

## Funcionalidades e Técnicas Aplicadas

* Texto com Gradiente Nativo: Aplicação de uma transição de cores fluida (passando por tons de roxo, violeta e rosa) que preenche o texto "GRADIENT" de ponta a ponta.
* Mascaramento de Fundo (Background Clipping): Uso avançado das propriedades `background-clip: text` e `color: transparent` para mascarar o gradiente linear dentro do formato dos caracteres.
* Tipografia de Forte Impacto: Escolha de uma fonte sem serifa, em caixa alta (*uppercase*) e com peso em negrito bem definido, garantindo a área de preenchimento ideal para evidenciar a transição de cores.
* Design Minimalista: Componente isolado sobre um plano de fundo puramente branco para assegurar o máximo contraste e destaque visual do efeito.
